```html
	j		向下滚动 (Scroll down)
	k		向上滚动 (Scroll up)

	G		滚动到页面的底部 (Scroll to the bottom of the page)

	u		向上滚动半页（Scroll a half page up）

	yy		将当前URL复制到剪贴板（Copy the current URL to the clipboard）
	p 		打开剪贴板的URL在当前选项卡（Open the clipboard's URL in the current tab）
	P		在新选项卡中打开剪贴板的URL（Open the clipboard's URL in a new tab）
	
	f		打开一个链接在当前选项卡（Open a link in the current tab）
	F		在新标签页中打开链接（Open a link in a new tab）
	yf		将URL的链接复制到剪贴板（Copy a link URL to the clipboard）

	o		打开网址,书签或历史条目（Open URL, bookmark or history entry）
	O		打开网址,书签或历史在新的选项卡（Open URL, bookmark or history entry in a new tab）
	H		回退（Go back in history）
	L		前进（Go forward in history）

	t		创建新选项卡（Create new tab）
	J, gT	打开左侧选项卡（Go one tab left）
	K, gt	打开右侧选项卡（Go one tab right）
	g0		第一个选项卡（Go to the first tab）
	g$		最后一个选项卡（Go to the last tab）
	x		关闭当前选项卡（Close current tab）
	X		恢复关闭的选项卡（Restore closed tab）
	
	W		移动选项卡到新窗口（Move tab to new window）
	<<		向左移动选项卡（Move tab to the left）
	>>		向右移动选项卡（Move tab to the right）

	?		显示帮助 （Show help）
	gs		查看页面源代码（View page source）
```

全部快捷键



```html
导航页面（Navigating the page）
	j		向下滚动 (Scroll down)
	k		向上滚动 (Scroll up)
	gg		滚动到页面的顶部 (Scroll to the top of the page)
	G		滚动到页面的底部 (Scroll to the bottom of the page)
	d		向下滚动半页（Scroll a half page down）
	u		向上滚动半页（Scroll a half page up）
	h 		向左滚动（Scroll left）
	l 		向右滚动（Scroll right）
	zH		滚动到左边（Scroll all the way to the left）
	zL		滚动到右边（Scroll all the way to the right）
	r 		请重新载入页面（Reload the page）
	yy		将当前URL复制到剪贴板（Copy the current URL to the clipboard）
	p 		打开剪贴板的URL在当前选项卡（Open the clipboard's URL in the current tab）
	P		在新选项卡中打开剪贴板的URL（Open the clipboard's URL in a new tab）
	gu		上升URL层级（Go up the URL hierarchy）
	gU		上升当前URL层级的根级（Go to root of current URL hierarchy）
	i		进入插入模式（Enter insert mode）
	v		进入视图模式（Enter visual mode）
	V		进入行视图模式（Enter visual line mode）
	gi		聚焦当前页的第一个文本输入（Focus the first text input on the page）
	f		打开一个链接在当前选项卡（Open a link in the current tab）
	F		在新标签页中打开链接（Open a link in a new tab）
	yf		将URL的链接复制到剪贴板（Copy a link URL to the clipboard）
	[[		跳转前一个链接标记（Follow the link labeled previous or <）
	]]		Follow the link labeled next or >
	gf		选择页面上的下一个框架（Select the next frame on the page）
	gF		选择页面的主/顶框架（Select the page's main/top frame）
	m		创建一个新的标志（Create a new mark）
	`		Go to a mark
	<a-f> 	Open multiple links in a new tab

Using the vomnibar
	o		打开网址,书签或历史条目（Open URL, bookmark or history entry）
	O		打开网址,书签或历史在新的选项卡（Open URL, bookmark or history entry in a new tab）
	b		打开一个书签（Open a bookmark）
	B		在新选项卡打开一个书签（Open a bookmark in a new tab）
	T		Search through your open tabs
	ge		编辑当前的URL（Edit the current URL）
	gE		编辑当前的URL并在新选项卡中打开（Edit the current URL and open in a new tab）

查找（Using find）
	/		进入查找模式（Enter find mode）
	n		循环到下一个匹配项（Cycle forward to the next find match）
	N		循环到前一个匹配项（Cycle backward to the previous find match）

浏览历史（Navigating history）
	H		回退（Go back in history）
	L		前进（Go forward in history）

操作选项卡（Manipulating tabs）
	t		创建新选项卡（Create new tab）
	J, gT	打开左侧选项卡（Go one tab left）
	K, gt	打开右侧选项卡（Go one tab right）
	^		去之前访问过的选项卡（Go to previously-visited tab）
	g0		第一个选项卡（Go to the first tab）
	g$		最后一个选项卡（Go to the last tab）
	yt		复制当前的标签（Duplicate current tab）
	<a-p>	Pin or unpin current tab	
	<a-m>	Mute or unmute current tab	
	x		关闭当前选项卡（Close current tab）
	X		恢复关闭的选项卡（Restore closed tab）
	W		移动选项卡到新窗口（Move tab to new window）
	<<		向左移动选项卡（Move tab to the left）
	>>		向右移动选项卡（Move tab to the right）

杂项 (Miscellaneous)
	?		显示帮助 （Show help）
	gs		查看页面源代码（View page source）
```
